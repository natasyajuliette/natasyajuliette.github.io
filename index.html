Reimbursement Application

Web-application that handles reimbursement requests and approvals at Christ the Cornerstone Church, Seattle, WA

A web application that utilizes REST API methods from church directory and reimbursement report microservices to retrieve church's members contact infomation for the user authentication and submission details and to store and update the church's reimbursement submission.

It consists of a form page to submit reimbursement requests and a dashboard page that show tables of reimbursement report submitted. The reimbursement form executes validation for each input that the submitter put in each label and also to check if the form is complete. The dashboard table shows tables of the user's reimbursement submission and if the user is an approver, shows tables of reimbursement reports submitted to the user to be reviewed.

I was in charge of implementing the REST API methods for the church’s directory and reimbursement report microservices using .Net Core SDK and C# and developing the church’s reimbursement form validation and report table using React, HTML, and CSS

  • Gitlab-2: reimbursement-report-git

Therejectedbot

A telegram bot to retrieve weekly schedule of Christ the Cornerstone Church Service

A telegram bot that allows the user (congregation) to retrieve information of weekly service at Christ the Cornerstone Church, Seattle, Washington, United States, and the people who serve by typing specific keywords.

/schedule to retrieve that week schedule

/schedule mm/dd/yyyy or mm/dd to retrieve that week schedule to the nearest Sunday specified by the date.

/schedule #weeknumber to retrieve that week schedule by the number of week specified.

/help to provide a list of features of therejectedbot Telegram bot

The bot is written in Python and the database is stored in Google Sheet and accessed using Python. It is developed by the Christ the Cornerstone Church Computer Programmers. I am one of the church programmers and added the /schedule mm/dd feature, /schedule #weeknumber, and /help feature.

  • Date: September 2017
  • Gitlab: therejectedbot-git
  • Link: therejectedbot
